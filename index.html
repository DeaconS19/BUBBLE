BUBBLE

Chrome v132 & Below | CVE-2025-0998 + CVE-2025-0447

Overview

BUBBLE is a two-stage Chrome exploit chain designed to demonstrate exploitation of Chromium v132 and earlier.

It leverages CVE-2025-0998 to achieve **initial code execution inside the browser sandbox**, and then escalates privileges using CVE-2025-0447, exploiting flaws in the navigation process to escape the sandbox.

Stage 1 – CVE-2025-0998

This vulnerability is an out-of-bounds memory access in the V8 JavaScript engine. By carefully crafting JavaScript that misleads the engine’s JIT assumptions, an attacker can write to restricted memory regions and execute arbitrary code within the sandboxed browser context.

Stage 2 – CVE-2025-0447

Once code execution is achieved, BUBBLE uses a privilege escalation via flawed navigation logic to break out of the sandbox. This allows the exploit to elevate privileges to the level of the Chrome process, potentially gaining access to host resources.
